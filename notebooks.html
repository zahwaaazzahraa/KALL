Pengertian Sistem Persamaan Linier#

Sistem Persamaan Linier (SPL) adalah kumpulan dua atau lebih persamaan yang harus dipenuhi secara bersamaan. Persamaan ini disebut linier karena setiap variabelnya berpangkat satu dan tidak ada perkalian antar variabel. Tujuan utama Sistem Persamaan Linier adalah mencari nilai variabel yang membuat semua persamaan benar.

tugas SPL#

Penyelesaian sistem persamaan linear menggunakan Metode Eliminasi Gauss adalah cara untuk menyelesaikan sistem persamaan dengan mengubahnya menjadi bentuk yang lebih sederhana menggunakan operasi baris elementer. Berikut adalah langkah-langkahnya:

Langkah 1: Menyusun Matriks Augmented#

Misalkan kita memiliki sistem persamaan linear berikut:

a1x + b1y + c1z = d1 a2x + b2y + c2z = d2 a3x + b3y + c3z = d3

Langkah pertama adalah menuliskan sistem persamaan ini dalam bentuk matriks augmented (gabungan antara koefisien dan hasil):

Matriks ini menggambarkan sistem persamaan linear dalam bentuk yang lebih mudah untuk dimanipulasi.

Langkah 2: Menggunakan Operasi Baris Elementer#

Operasi baris elementer adalah operasi yang digunakan untuk memanipulasi matriks augmented dengan tujuan untuk mendapatkan bentuk yang lebih sederhana. Tiga jenis operasi yang diperbolehkan adalah:

  1. Menukar dua baris.

  2. Mengalikan baris dengan suatu bilangan bukan nol.

  3. Menambahkan kelipatan dari satu baris ke baris lain.

Dengan operasi ini, kita akan mengubah matriks augmented menjadi bentuk baris eselon, di mana setiap baris di bawah baris utama memiliki lebih banyak nol, sehingga memungkinkan kita untuk menyelesaikan sistem persamaan.

Langkah 3: Menyusun Matriks ke dalam Bentuk Eselon Baris#

Langkah ini bertujuan untuk mengubah matriks ke dalam bentuk yang disebut bentuk eselon baris atau row echelon form (REF), di mana:

  1. Baris yang hanya berisi nol berada di bagian bawah matriks.

  2. Koefisien pertama dari setiap baris (pivot) harus lebih besar dari nol dan terletak lebih jauh ke kanan pada setiap baris berikutnya.

Contoh jika kita mulai dari matriks augmented seperti berikut:

1 2 1 3 2 4 1 5 3 1 2 7

Kita akan menggunakan operasi baris elementer untuk mendapatkan bentuk eselon baris. Misalnya, kita ingin menghilangkan elemen-elemen di bawah pivot pertama (1 di kolom pertama).

Langkah pertama, kita kurangi baris 2 dengan 2 kali baris 1, dan kurangi baris 3 dengan 3 kali baris 1, sehingga menjadi: 1 2 1 3 0 0 -1 -1 0 -5 -1 1

Kemudian, kita akan beralih ke kolom kedua dan melanjutkan operasi hingga mendapatkan bentuk yang lebih sederhana.

Langkah 4: Mengubah ke Bentuk Eselon Baris Tereduksi#

Setelah kita mendapatkan bentuk eselon baris (REF), langkah berikutnya adalah mengubahnya ke bentuk eselon baris tereduksi (RREF), di mana setiap elemen pivot menjadi 1 dan semua elemen di atas pivot menjadi 0.

Misalnya, kita akan lanjutkan operasi untuk mengubah matriks menjadi: 1 0 0 x 0 1 0 y 0 0 1 z

Di sini, kita akan menemukan nilai dari variabel x, y, z

Langkah 5: Menyelesaikan Sistem#

Setelah matriks berada dalam bentuk eselon baris tereduksi, kita dapat membaca solusi langsung dari matriks. Jika sistem tersebut memiliki solusi unik, maka solusi dari setiap variabel dapat ditemukan langsung dari matriks. Misalnya, jika kita memiliki bentuk: 1 0 0 2 0 1 0 3 0 0 1 4

Maka solusi sistem persamaan tersebut adalah: x = 2, y = 3, z = 4

Langkah 6: Menyimpulkan Solusi#

Diketahui:#

x + y + z = 6

2x - y + 3z = 14

3x + 2y - z = 2

Metode eliminasi Gauss ini berguna untuk sistem persamaan linear yang melibatkan lebih dari dua variabel, karena memungkinkan kita menyelesaikan sistem persamaan secara terstruktur dan sistematis.
